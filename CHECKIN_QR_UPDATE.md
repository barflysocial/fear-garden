# Check-In QR Update

Added venue QR-code check-in.

Host dashboard:
- Each session now shows a Venue Check-In QR code.
- Host can copy the check-in link or open the check-in page.
- Check-in does not start the game.

Player check-in page:
- Players scan the venue QR code.
- They enter first name, last name, optional Instagram/contact, and access code.
- The app verifies the access code belongs to that session.
- Paid event codes must be activated/paid before check-in.
- Free event shared codes can check in multiple players until capacity is reached.
- The RSVP/check-in list updates on the host dashboard.
