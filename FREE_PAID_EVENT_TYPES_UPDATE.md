# Free/Paid Event Types Update

Added host-selectable event types:

- Free Event: one shared access code for everyone in that session.
- Paid Event: one unique activated access code per player.

Host dashboard changes:

- Added Event Type selector.
- Added Shared Code for Free Event field.
- Free sessions show a shared access code card instead of a unique-code table.
- Paid sessions continue to show unique player access codes.
- RSVP rows adapt based on event type.

Player RSVP changes:

- Showtime cards show Free Event or Paid Event.
- Free RSVP confirmations return the shared access code.
- Paid RSVP confirmations tell the player they will receive a personal code after payment.

Access behavior:

- Free event shared codes can be used by multiple players until capacity is reached.
- Paid event codes must be activated/marked paid by the host before entry.
- Paid event codes can only be claimed once.
- Demo code behavior is unchanged and remains hidden from the player UI.
