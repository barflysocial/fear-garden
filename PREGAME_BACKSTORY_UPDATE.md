# Pregame Backstory Update

Added a 5-minute pre-game case setup for Fear Garden. When the host starts a session, or when Demo Mode begins, the player now sees a cinematic Case Briefing screen with its own 5:00 countdown. The 30-minute investigation timer does not begin during this setup period. When the 5-minute countdown expires, the server automatically starts the investigation and the normal 30-minute gameplay timer begins.
