IMPORTANT RENDER UPLOAD NOTE

Do NOT upload this ZIP file itself into GitHub.

1. Download this ZIP.
2. Extract/unzip it on your computer.
3. Open the extracted folder.
4. Upload the CONTENTS of the folder to GitHub so package.json is visible at the top/root of the repo.

Your GitHub repo root must show:
- package.json
- server.js
- database.js
- public/
- truth-packs/
- data/
- render.yaml

Render settings:
Build Command: npm install
Start Command: npm start
Root Directory: leave blank unless you intentionally placed these files inside a subfolder.
